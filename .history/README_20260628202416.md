# BlockLang

Visual programming for learning real TypeScript — typed puzzle blocks that compile to code and run step-by-step.

![BlockLang editor](./src/assets/screenshot.png)

## About

BlockLang helps beginners understand programming fundamentals through a visual block editor. Concepts map directly to TypeScript: data types, variables, expressions, functions, and control flow. The MVP focuses on clarity over breadth.

## How it works

- **Main Function** — every program starts here; it cannot be deleted or nested
- **Typed slots** — blocks only attach where types match; invalid drops show clear errors
- **Block editors** — nested blocks appear as mini chips; click to edit in a stacked panel
- **TypeScript output** — blocks compile to TypeScript shown in the Inspector
- **Emulate** — run the program with live variable state and execution highlighting

## Core rules

- Everything is a block; typed value slots reject mismatches
- Constants coerce to the slot's expected type on attach
- Nested blocks always use mini view; full UI is for top-level and editor roots
- Functions declare a return type; use Return blocks in the body to send values back

## Tech stack

React, TypeScript, Vite
